# Kinesis

ストリーミング処理用のサービス。

## Data Streams

データ処理プログラムを構築するサービス。処理の中心、APサーバーみたいなやつ。

**シャード**単位で処理するため、高速な並列処理が可能。シャードにインスタンスを割り当てる。

## Data Firehouse

データをS3やRedShiftへ配信。
